# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the AEM Edge Delivery Services + Adobe App Builder programme.

ADRs are proposed by any team member and **approved by the AEM Technical Architect**.

---

## Format

Each ADR follows the format `NNN-short-title.md` and contains the following sections:

| Section | Description |
|---|---|
| **Status** | `Proposed` → `Accepted` → `Deprecated` / `Superseded` |
| **Context** | The situation or problem that drove the decision |
| **Decision** | The choice that was made |
| **Consequences** | Trade-offs, follow-on actions, constraints introduced |
